# La Casa de Papel - Proyecto Web Escolar

Sitio web escolar de una sola página dedicado a la serie **La Casa de Papel**, construido con HTML5, CSS3 y Bootstrap 5.

## 📌 Descripción

El proyecto tiene una página principal (`index.html`) que presenta información de la serie, un formulario de registro y un video integrado. Además, el sitio incluye un menú de navegación que lleva a una segunda página con tarjetas de personajes.

Cada tarjeta muestra la imagen de un integrante de la serie y enlaza a una página interna con información detallada sobre ese personaje.

## 🎯 Objetivo

Desarrollar una experiencia web clara, ordenada y responsiva para presentar contenidos de la serie y facilitar la navegación entre la página principal, el formulario y las fichas de los personajes.

## 🧩 Funcionalidades principales

- Página principal con presentación de la serie.
- Formulario de registro para los usuarios.
- Video integrado dentro de la página principal.
- Menú de navegación que dirige a una segunda página de personajes.
- Tarjetas de personajes con imagen y enlace a la ficha correspondiente.
- Páginas internas con información de cada personaje.

## 📁 Estructura del proyecto

- `index.html` — Página principal con información, formulario y video.
- `CSS/bootstrap.min.css` — Estilos de Bootstrap 5.
- `CSS/styles.css` — Estilos personalizados.
- `imagenes/` — Recursos gráficos y visuales.
- `pages/` — Página de personajes y páginas individuales por personaje.

## 🚀 Cómo usar

1. Abrir `index.html` en el navegador.
2. Usar el menú para acceder a la página de personajes.
3. Hacer clic en cualquier tarjeta para ver la ficha del personaje.

## 🛠️ Tecnologías usadas

- HTML5
- CSS3
- Bootstrap 5

## ✅ Beneficios del proyecto

- Diseño uniforme y profesional.
- Navegación intuitiva entre secciones.
- Contenido organizado en una sola página principal y fichas internas.
- Mejoras en semántica HTML y limpieza del CSS.
